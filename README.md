# mqtt-graphql-iot
This repository provides a GraphQL Endpoint based on Apollo Server and Express Middleware to subscribe to live IoT Sensor Data publishing to a MQTT Broker.

## IoT Part
Hardware:

* ESP32
* BME280

Software:

* Arduino v1.8.9
* MQTT Broker (public/private)
